# FitMe Frontend 🎨

> 대학생 맞춤형 장학금 · 공모전 매칭 서비스 — 프론트엔드

## 기술 스택

| 구분 | 기술 |
|------|------|
| Framework | React |
| Style | CSS / Tailwind |
| Platform | 모바일 웹 |

## 브랜치 전략

- `main` : 최종 배포
- `develop` : 통합 개발
- `feat/frontend` : 기능 개발 → develop으로 PR

## 시작하기

```bash
npm install
npm start
```
